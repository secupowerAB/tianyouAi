# AI英语作文生成器

## 项目简介

本项目是一个基于人工智能的英语作文生成工具，旨在帮助高中学生练习和学习英语写作。通过输入主题和要求，系统可以自动生成符合要求的英语作文，供学生参考和学习。

## 使用方法

### 1. 准备工作
1. 访问 [Anthropic控制台](https://console.anthropic.com) 注册账号
2. 创建API密钥（sk-ant-开头）
3. 保存好你的密钥

### 2. 使用步骤
1. 打开 `essay-generator.html` 文件
2. 在"API密钥配置"区域输入你的API密钥
3. 输入作文主题（例如："The Importance of Environmental Protection"）
4. 选择作文类型和字数要求
5. 可选：添加补充说明
6. 点击"生成作文"按钮
7. 等待几秒钟，作文将显示在结果区域
8. 点击"复制作文"按钮可以复制全文

### 3. API密钥获取
- 访问：https://console.anthropic.com
- 注册并登录账号
- 进入 API Keys 页面
- 点击 "Create Key" 创建新密钥
- 复制密钥并妥善保存

## 安装部署

### 本地使用
1. 下载项目文件
2. 双击打开 `essay-generator.html`
3. 在浏览器中使用（推荐Chrome、Edge、Firefox）

### GitHub Pages部署
1. 创建GitHub仓库
2. 上传 `essay-generator.html` 文件
3. 在仓库设置中启用GitHub Pages
4. 访问生成的网址即可使用

详细部署步骤请参考《安装部署指南.md》

## 注意事项

### API使用
- API密钥仅保存在浏览器本地
- 每次生成会消耗少量API配额
- 建议合理使用，避免频繁调用

### 学习用途
- 本工具生成的作文仅供学习参考
- 建议学生理解内容后进行改写
- 不建议直接提交为作业

### 浏览器兼容性
- 推荐使用现代浏览器（Chrome 90+、Edge 90+、Firefox 88+）
- 需要启用JavaScript
- 需要联网（调用API）
